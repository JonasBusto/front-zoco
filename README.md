# Frontend ZOCO

Frontend hecho con React, Typescript, ContextAPI y custom hooks, con JSON-Server para simular un backend.

_URL:_ Debido a que use JSON-SERVER, mis unicas opciones eran render o railway, ya que requieren configuraciones especificas para el json, pero no pude subir el proyecto por superar el limite gratuito. Si fuera un backend y frontend separados (lo cual no cumple con los requisitos solicitados) podria subirlos directamente a Vercel. De igual forma, estan las instrucciones para ejecutarlo localmente. üôÇüëç

## _Utilice:_

- **JSON-Server:** Para simular un backend.
- **React Router Dom:** Para navegar entre paginas.
- **Formik:** Para validar los fomularios.
- **Prime React:** Para las tablas del administrador.
- **Axios:** Para el consumo de API.
- **Context:** Para gestionar estados globales.
- **Tailwind:** Para los estilos.
